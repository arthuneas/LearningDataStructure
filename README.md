## Sobre o Repositório

Este espaço contém implementações em C de várias estruturas de dados fundamentais e algoritmos associados, desenvolvidos como parte de estudos acadêmicos na Universidade de Brasília (UnB). O objetivo é fornecer exemplos claros e eficientes de como essas estruturas funcionam no hardware, priorizando o controle direto sobre a memória e a eficiência.

## Estruturas Incluídas

O repositório está organizado em pastas para cada tipo de estrutura. Todo o código foi escrito do zero:

- **Listas (Lists)**
- **Filas (Queues)**
- **Pilhas (Stacks)**
- **Algoritmos de Ordenação (Sorting)**
- **Árvores (Trees)**
- **Grafos (Graphs)**
